
# ----- Descrição de cada arquivo -----

1) about.qmd - script

2) forest.Rproj - Arquivo do RStudio Project. Guarda preferências do projeto (diretório de trabalho, opções de console, build tools, Git integrado etc.).
	Não é “código” do site; é só para facilitar seu trabalho no RStudio.
	
3) index.qmd - Página principal do site. É um documento Quarto (Markdown estendido) que vira index.html quando você renderiza.
	Conteúdo típico: apresentação, figura, links, seções principais.

4) _quarto.yml - Arquivo de configuração do Quarto (YAML). Define coisas como título do site, navegação, tema, pastas de saída, formato HTML/PDF, etc.

5) README.md - Descrição do repositório no GitHub (aparece na página inicial do repo).

É independente do site; serve para orientar quem vê o código (como instalar, renderizar, citar, etc.).
	Não substitui index.qmd (que é a home do site).

6) styles.css - Folha de estilo para personalizar o visual (cores, fontes, espaçamentos).
	Você referencia esse CSS no _quarto.yml.
